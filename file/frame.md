# 🚀 資訊班年度成果發表會 - 網頁建置企劃書

此網頁旨在展示資訊班學員的技術實力，同時透過「小肥啾」吉祥物營造親和且具溫度的視覺氛圍。

---

## 🎨 核心設計理念 (Design Vision)

* **視覺風格**：清新治癒 (Light) 與 極客深夜 (Dark/Coding Night) 雙模式切換。
* **吉祥物**：以「小肥啾」作為引導員（Guide），貫穿整個用戶體驗（UX）。
* **技術感**：融入雷達圖、Git 提交統計、Tech Tags 等開發者元素。

---

## 🛠️ 全站通用功能 (Global Features)

1. **Loading 動畫 (The Builder Bird)**
   - 頁面載入時，顯示小肥啾叼著木頭（或代碼符號）蓋房子的動畫。
2. **深色模式 (Dark Mode Switch)**
   - 右上角提供切換開關。
   - 淺色模式：小肥啾配色（極光白、晚霞粉）。
   - 深色模式：Coding Night 氛圍（墨黑背景、螢光色標籤）。
3. **流暢捲動 (Smooth Scrolling)**
   - 採用 GSAP 或 AOS 實作內容浮現動畫，提升質感。

---

## 🌐 網頁架構 (Sitemap)

### 1. 首頁 (Home - Landing Page)
* **視覺焦點**：大標題「Code with Heart」搭配小肥啾探頭互動。
* **活動快訊**：即時倒數計時器（距離發表會開始時間）。
* **CTA 按鈕**：[立即報名]、[專題搶先看]。

### 2. 關於我們 (Class Intro)
* **班級故事**：描述從程式小白到開發者的轉變。
* **技術總覽**：用橫向捲動展示班級共同學習過的技術圖標。
* **團隊合照**：懸停(Hover)時照片會從正式照變為搞怪照。

### 3. 節目表 (Program / Schedule)
* **動態時間軸**：垂直時間軸設計，標註各階段活動。
* **吉祥物位置**：小肥啾會在當前進行中的時間點旁邊顯示「We are here!」。

### 4. 專題介紹 (Project Showcase)
* **篩選功能**：依技術標籤（Python, React, AI...）分類。
* **專題卡片內容**：
    * **封面圖**：Demo GIF 或 專題預覽圖。
    * **技術棧 (Tech Stack Tags)**：使用醒目膠囊型標籤。
    * **實力證明**：顯示該組的 Git 提交次數或核心邏輯圖。
    * **互動按鈕**：[查看作品細節]、[GitHub 連結]。

### 5. 成員介紹 (Meet the Team)
* **互動式名片**：
    * **正面**：Q 版頭像（可參照小肥啾風格）、姓名、技術定位。
    * **反面 (Hover 彩蛋)**：
        * **技能雷達圖**：視覺化個人的開發能力分佈。
        * **開發語錄**：分享開發期間「最崩潰的 Bug」或「最自豪的一行 Code」。

### 6. 活動報名 (Registration)
* **報名表單**：簡潔的 UI，填寫完成後觸發「小肥啾撒花」特效。
* **成功頁面**：生成專屬的電子門票小卡。

### 7. 【發表後啟動】即時互動區 (Live Interaction)
* **投票系統**：成發開始後開啟，參觀者可點擊愛心為心儀專題投票。
* **互動留言板**：留言會以彈幕或瀑布流形式出現在網頁下方，增強臨場感。

### 8. 花絮與紀錄 (Highlights & Records)
* **回憶照片牆**：使用瀑布流佈局展示爆肝開發、課堂互動照片。
* **成果紀錄**：嵌入當天直播錄影與各組最終 PPT 下載。

---

## 📈 技術規格建議 (Technical Stack)

* **前端框架**：HTML5, CSS3 (SCSS), JavaScript (ES6+)。
* **動畫庫**：GSAP (用於複雜動畫) 或 AOS (用於簡單浮現)。
* **圖表庫**：Chart.js (用於成員技能雷達圖)。
* **字體**：選用無襯線且具現代感的字體（如 Noto Sans TC）。
